Comprehensive WooCommerce toolkit for bulk editing, PDF invoices, and order management with local processing.

== Description ==

AdminX WooCommerce Toolkit is a powerful plugin that enhances your WooCommerce store management with essential tools for bulk editing, PDF invoice generation, and comprehensive activity logging. All features run locally without external dependencies.

= Key Features =

**Bulk Price & Stock Editor**
* Bulk update product prices with multiple actions (set, increase, decrease, percentage changes)
* Bulk manage stock quantities and stock status
* Bulk edit product categories and tags
* Advanced filtering and selection options
* Undo functionality for bulk operations

**Local PDF Invoice Generator**
* Automatic invoice generation on order completion
* Manual invoice generation from order pages
* Customizable invoice templates
* Company branding and information
* Local PDF generation using TCPDF or dompdf
* Download and email invoices

**Order & Customer Activity Log**
* Track all WooCommerce activities (orders, products, customers)
* User login/logout tracking
* Detailed change logs with before/after values
* IP address and user agent logging
* Exportable activity reports
* Configurable log retention periods

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/adminx-woo-toolkit/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to AdminX WooCommerce in your admin menu
4. Configure settings according to your needs
5. Start using the bulk editor, invoice generator, and activity log

= Requirements =

* WordPress 5.0 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin is specifically designed for WooCommerce and requires WooCommerce to be installed and active.

= Are there any external dependencies? =

No, all features work locally on your server. No external API calls or third-party services are required.

= Can I customize the PDF invoice template? =

Yes, the plugin includes customizable invoice templates. You can modify company information, branding, and layout through the settings.

= How long are activity logs stored? =

By default, activity logs are stored for 90 days. You can configure this retention period in the plugin settings.

= Can I export activity logs? =

Yes, you can export activity logs to CSV format with filtering options for date ranges, users, and activity types.

= Is the bulk editor safe to use? =

Yes, the bulk editor includes confirmation dialogs and detailed logging of all changes. However, we recommend backing up your database before performing large bulk operations.

== Privacy Policy ==

This plugin stores activity logs locally in your WordPress database. No data is sent to external servers. Activity logs include user actions, IP addresses, and user agent strings for security and auditing purposes. You can configure log retention periods and export/delete logs as needed.
